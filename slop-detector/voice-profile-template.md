# Voice Profile

This file does not exist by default. Create it when you have enough samples of the user's writing to characterize their voice. See `references/voice-strategy.md` for the full process.

To create your voice profile, give the agent 3-5 samples of writing you're proud of and ask: "Build my voice profile from these."

## Template

```markdown
# Voice Profile: [Name]

## Rhythm
[Sentence length patterns, paragraph habits, pacing]

## Stance
[Confident? Understated? Direct? Self-deprecating? Observational?]

## Signature moves
[2-3 specific habits unique to this writer]

## Avoids
[Things this writer clearly never does]

## Register
[Default formality level, when it shifts]

## Sample sentences
[3-5 real sentences that exemplify the voice]
```
