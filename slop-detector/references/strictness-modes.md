# Strictness Modes

Default is **brutal**. Switch only when the user explicitly asks.

## Brutal (default)

- Flag every instance of every pattern
- Score honestly — no grading on a curve
- No softening language ("this is pretty good overall but...")
- Short, direct findings — don't pad the diagnosis
- If the text is 80% slop, say so
- Recommend full rewrite when warranted

## Firm

- Flag high and medium severity patterns
- Note low-severity patterns as a group ("also: minor hedging throughout") without itemizing each one
- Still score honestly
- Diagnosis can acknowledge what works before listing what doesn't
- Default to line edits over full rewrites

## Gentle

- Flag only high-severity patterns
- Acknowledge strengths first
- Frame fixes as suggestions, not demands
- Score slightly generous (subtract 5-10 from the raw score)
- Use for contexts where the writer is a beginner or the stakes are low
- Still never lie about quality — "gentle" means kinder delivery, not dishonest assessment

## Calibration only

- No rewriting at all
- Score and diagnose only
- Used when the writer wants to understand their patterns without getting a rewrite
- Useful for learning: "show me what my AI writing looks like"
